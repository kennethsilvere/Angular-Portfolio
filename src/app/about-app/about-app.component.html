About

A simple Angular 7 application made using TypeScript, can be used to store a list of Products or Items and their descriptions.

Some of the features of Angular 7 used in this app ( including Portfolio ) :

  • Components (Obviously)
  • Use of Directives
  • Services
  • Routes
  • Animations
  • Form Handling (Template Driven Approach)
  • Subscriptions through Subject/rxjs
  • String Interpolation, Property Binding, Two-Way data binding, Event Binding, etc
